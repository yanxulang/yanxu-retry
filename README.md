# 言韧

言韧（`yanxu-retry`）是言序的重试、指数退避和断路器辅助库。策略与执行分离，因而业务代码可以在测试中跳过真实等待，同时验证完全相同的次数和退避逻辑。

```yanxu
引「包:言韧」为 言韧；

法 请求一次（次数：数）则
    # 调用可能失败的操作
终

定 结果 为 言韧.执行（言韧.默认策略（），请求一次，言韧.总是重试）；
```

还提供三态断路器 `断路器`，具备连续失败阈值、冷却期、半开探测、成功复位与错误透传。错误使用 `YANREN_` 前缀。

```sh
yanxu 查 src/言韧.yx
yanxu 试 tests
yanxu 字节 tests/重试.yx
```

公共接口见 [docs/API.md](docs/API.md)。本项目采用 MIT 许可证。
