# 言韧 API

- `重试策略（最大次数，初始延迟秒，倍率，最大延迟秒）`
- `默认策略（）：重试策略`
- `立即策略（最大次数）：重试策略`
- `延迟秒`、`延迟表`
- `执行（策略，操作，判定）`：按退避表真实等待。
- `立即执行（策略，操作，判定）`：不等待，方便测试或外部调度。
- `尝试执行（策略，操作，判定，真实等待）：典`：将最终错误转成结构化结果。
- `总是重试`、`从不重试`：内建判定器。

操作接收从 1 开始的尝试次数；判定器接收捕获到的 `误` 与本次次数。

`断路器` 提供 `状态`、`是否允许`、`记成功`、`记失败`、`失败数` 与 `调用`。
